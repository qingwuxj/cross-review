# Historical Implementation Plans

This directory contains archived planning artifacts from Cross-Review development. They are useful for contributors who want to audit how a feature was designed or why a tradeoff was made.

These files are not current user documentation and should not be read as open issues, release requirements, or required setup steps. For current usage, use `README.md`, `SKILL.md`, `docs/README.md`, and `docs/implementation-roadmap.md`.
